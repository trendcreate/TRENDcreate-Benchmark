使い方 / How to use

自分のPCでベンチを実行 → できた score/*.json を下の「スコアを共有」にドロップして Issue 送信すると、このランキングに自動で反映されます。外部ライブラリ不要・Python3だけで動きます。

Linux / macOS（コピペ1行）

curl -fsSL https://raw.githubusercontent.com/trendcreate/TRENDcreate-Benchmark/main/Tc_bench.py -o Tc_bench.py && python3 Tc_bench.py

Windows（PowerShell）

curl.exe -fsSL https://raw.githubusercontent.com/trendcreate/TRENDcreate-Benchmark/main/Tc_bench.py -o Tc_bench.py
py Tc_bench.py

算出方法について

スコアは他マシンと相対比較するための独自指標です。CPU は実際に重い計算（素数判定＋浮動小数点演算）を 回した実測時間から算出するため、オーバークロック等の成果はそのまま反映されます。 GPU は計測負荷をかけず、スペック値から算出する「ロマン枠（V-Calc）」です。 

CPU Single    = 10000 / シングル実行秒数
CPU Multi     = 10000 × 論理スレッド数 / マルチ実行秒数
GPU V-Calc    = 最大クロック(MHz) × (VRAM_GB ^ 1.5) / 10   ※クロック不明時は公称 1500MHz
GPU Diffusion = デノイズ steps/sec × 100   ※plugin (要 PyTorch GPU版 / VRAM 4GB+空き2GB)
  • CPU Single … 1コアの実測性能（速い＝高スコア）
  • CPU Multi … 全論理スレッドを使った実測性能
  • GPU V-Calc … VRAM 容量を重視した仮想スコア（NVIDIA / AMD 対応、未検出ならスキップ）

計測はアイドル状態を推奨。電源/温度による周波数制御や同時実行プロセスでスコアは変動します。

⚠️ 偽造JSONの投稿について

公式ランキングには実測の正規スコアのみ投稿してください。改ざん・偽造した JSON を [score] Issue で投稿しないでください。
