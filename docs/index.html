スコアを投稿する手順

  1. このリポジトリを Fork する
  2. あなたの score/*.json を score/ に追加
  3. python tools/build_pages.py を実行して docs/data.json を更新
  4. Pull Request を送る → マージされるとこのボードに反映されます

手早く共有したいだけなら Issue を開いて JSON を貼る でもOKです。
